# 🐦 Flappy Bird Java

Uma recriação do clássico **Flappy Bird** desenvolvida em **Java** utilizando **Swing** e **AWT**, com foco em conceitos fundamentais de desenvolvimento de jogos 2D, manipulação de eventos, colisões e renderização gráfica.

---

## 📸 Visão Geral

Neste jogo, o jogador controla um pássaro que deve atravessar uma série de canos sem colidir com eles ou com o chão. A cada obstáculo superado, a pontuação aumenta, tornando o desafio cada vez mais emocionante.

### Principais características

✅ Interface gráfica desenvolvida com Java Swing

✅ Sistema de física simples com gravidade e impulso

✅ Detecção de colisões em tempo real

✅ Geração aleatória de obstáculos

✅ Sistema de pontuação

✅ Reinício rápido após Game Over

✅ Controles simples e responsivos

---

## 🎮 Como Jogar

### Objetivo

Passe pelo maior número possível de canos sem colidir.

### Controles

| Tecla             | Ação                 |
| ----------------- | -------------------- |
| SPACE             | Fazer o pássaro voar |
| SPACE (Game Over) | Reiniciar partida    |

### Regras

* Evite bater nos canos.
* Não deixe o pássaro cair.
* Cada conjunto de canos ultrapassado aumenta sua pontuação.
* Tente alcançar o maior score possível.
